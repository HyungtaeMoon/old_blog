<br/>
파이썬3에서는 문자열에서 유니코드를 사용
<br/>
**인덱싱**
<br/>
```python
>>> welcome = '환영합니다 이웃님!'
>>> welcome[0]
>>> '환'
>>> welcome[3]
```
<br/>
<br/>
**슬라이싱**
문자열 **환영합니다** 만 추출하기
```python
>>> welcome = '환영합니다 이웃님!'
>>> welcome[0:5]
>>> '환영합니다'
```
