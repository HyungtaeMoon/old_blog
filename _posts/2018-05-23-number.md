수학연산자
<br/>
<br/>
산술연산자의 결합
<br/>
```python
a = 100
a = a - 3
print(a)
```
<br/>
산술연산자를 사용하면 아래와 같이 사용 가능
<br/>
```python
a = 100
a -= 3
print(a)
```
**위 2개의 변수는 같은 결과값이 출력된다**
```
a = a - 3
a -= 3
은 정확히 동일한 계산식이다
```
