<br/>
파이썬은 모든 것(정수, 문자열, 함수 등)이 **객체(object)** 이다.
<br/>
<br/>
<br/>

**객체지향과 절차지향의 차이는?**
```
1 ~10 숫자 중에 6을 선택하시오.
객체지향: 바로 6을 선택
절차지향: 1, 2... 5, 6 선택
```
<br/>
##변수는 단지 이름이며,
<br/>
##지정한 객체를 직접 가지는 것이 아니다.
<br/>
```
var1 = 100
```
var1 에 100 이 포함되었다는 의미가 아닌 **100을 가리키고 있다** 는 뜻이다.
<br/>
<br/>
<br/>
변수명은 예약어를 사용할 수 없다.
```
def = 100
```
```
zsh: command not found: def
```
def 라는 예약어를 변수로 지정하니 **잘못된 모드** 를 지정했다고 에러 메시지가 출력된다.
