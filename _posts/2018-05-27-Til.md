클래스 분석하기(수업내용)
```python
class User:
    def __init__(self, name):
        self.name = name
        self.money = 0

    def add_money(self, amount):
        self.money += amount
        print(f'소지금이 {amount}만큼 증가하였다!')

    def sell_item(self):
        self.add_money(100)

    def kill_monster(self):
        self.add_money(50)
```
